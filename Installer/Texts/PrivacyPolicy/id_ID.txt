KEBIJAKAN PRIVASI:
1. Pendahuluan
Selamat datang di Bedrock Cosmos. Kami menghormati privasi Anda dan berkomitmen untuk melindungi informasi pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan menjaga informasi Anda saat Anda menggunakan peluncur dan layanan kami.

2. Informasi yang Kami Kumpulkan
Bedrock Cosmos tidak mengumpulkan, menyimpan, atau mengirimkan informasi pribadi atau data penggunaan apa pun dari pengguna. Peluncur beroperasi sepenuhnya sebagai proksi sisi klien di perangkat lokal Anda dan tidak mengirimkan informasi apa pun ke server kami.

Secara khusus, kami TIDAK mengumpulkan:

- Informasi pribadi (nama, email, alamat, dll.)
- Data penggunaan atau analitik
- Informasi teknis tentang perangkat Anda
- Informasi akun Minecraft Anda
- Data gameplay atau interaksi apa pun

Pengiriman Kosmetik Kustom: Jika Anda secara sukarela memilih untuk mengirimkan desain jubah atau skin kustom melalui server Discord kami untuk dimasukkan ke dalam Bedrock Cosmos, kami akan mengumpulkan dan menyimpan file jubah tersebut beserta nama pengguna Discord Anda untuk mengidentifikasi Anda sebagai pembuatnya. Ini sepenuhnya opsional dan hanya terjadi ketika Anda secara aktif mengirimkan konten dalam komunitas kami. Jubah dan skin yang dikirimkan dapat tersedia untuk semua pengguna Bedrock Cosmos melalui peluncur.

Jika Anda bergabung dengan server Discord kami untuk dukungan atau interaksi komunitas, kebijakan privasi Discord sendiri berlaku untuk interaksi tersebut. Kami mungkin melihat nama pengguna Discord Anda dan konten apa pun yang Anda bagikan secara sukarela di server, tetapi ini terpisah dari peluncur Bedrock Cosmos itu sendiri.

3. Cara Kami Menggunakan Informasi Anda
Karena Bedrock Cosmos tidak mengumpulkan informasi pengguna apa pun dari peluncur itu sendiri, tidak ada data pribadi yang perlu kami gunakan, bagikan, atau proses. Peluncur berfungsi sepenuhnya di perangkat Anda sebagai proksi untuk memodifikasi data antara klien Minecraft Anda dan server Minecraft.

Semua pemrosesan terjadi di perangkat Anda, termasuk:

- Menambahkan jubah kustom ke pengalaman game Anda
- Mengungkapkan konten marketplace tersembunyi untuk wilayah Anda
- Memulihkan fitur yang hilang pada paket skin (jubah, animasi, skin)

Pengiriman Kosmetik Kustom: Jika Anda mengirimkan desain jubah atau skin kustom melalui server Discord kami, kami dapat menggunakan file tersebut untuk membuatnya tersedia bagi semua pengguna Bedrock Cosmos melalui peluncur. Kosmetik yang Anda kirimkan menjadi bagian dari koleksi komunitas bersama jika diterima.

4. Penyimpanan dan Keamanan Data
Bedrock Cosmos tidak mengumpulkan atau menyimpan data pengguna apa pun melalui peluncur. Peluncur beroperasi sebagai proksi lokal di perangkat Anda, mencegat dan memodifikasi lalu lintas jaringan antara klien Minecraft Anda dan server Minecraft untuk menyediakan jubah dan paket skin kustom, mengungkapkan konten marketplace tersembunyi, dan memulihkan fungsionalitas paket skin.

Pengiriman Kosmetik Kustom: Desain jubah dan skin kustom yang Anda kirimkan secara sukarela melalui server Discord kami disimpan dalam database kami beserta nama pengguna Discord Anda untuk atribusi pembuat. Hal ini memungkinkan kami dan pengguna lain untuk mengidentifikasi siapa yang membuat setiap desain. File yang dikirimkan sendiri tidak mengandung informasi pribadi tambahan selain nama pengguna Discord Anda.

Bedrock Cosmos dapat berinteraksi dengan layanan pihak ketiga:

- Discord: Untuk dukungan komunitas dan keterlibatan pengguna
- GitHub: Untuk menghosting dan mendistribusikan kode sumber dan rilis aplikasi

Layanan pihak ketiga ini memiliki Kebijakan Privasi mereka sendiri. Kami mendorong Anda untuk meninjau kebijakan mereka, karena kami tidak bertanggung jawab atas praktik privasi mereka.

Bedrock Cosmos tidak menggunakan cookie, teknologi pelacakan, atau alat analitik. Peluncur tidak melacak aktivitas Anda, mengumpulkan statistik penggunaan, atau memantau perilaku Anda dengan cara apa pun.
